JS

JS是一门脚本语言。

Java和JavaScript没有没和关系

一个合格的后端人员，必须精通JavaScript

ESMAScript是JavaScript的一个标准，最新版本是ES6，但是但部分浏览器还只支持ES5，

因此开发环境（ES6）和线上环境（ES5）的版本不一致。我们需要借助代码解决这兼容性问题。

引入JS的3种方式

注意：

<script>标签必须成对出现

type="text/javascript"可以不用显式定义，默认就是。

内部标签

test.html

<script type="text/javascript">
    alert("hello,JavaScript");
</script>

外部引入

hello.js

alert("hello,JS");

test.html

<script type="text/javascript" src="hello.js"></script>

定义变量（var）

i =1;    //全局变量

var a = 1;
var 王者荣耀;
var 22SS;    //错误，Uncaught SyntaxError: Invalid or unexpected token

严格检查模式

由于变量定义的太过随意，因此推出严格检查模式（前提：开启ES6规范）

在代码中加入use strict：表示开启严格检查模式，防止Javascript的随意性导致产生一些问题。

use strict必须写在第一行：

  • 不写在第一行：IDE报错，console不报错
  • 写在第一行：IDE报错，console报错

局部变量建议都使用let关键字定义。

数据类型

(1) number —— 数字

JS不区分小数和整数，统一称为number类型

console.log(typeof 123);        // number，整数
console.log(typeof 123.1);        // number，小数
console.log(typeof 123e3);        // number，科学计数法
console.log(typeof -123);        // number，负数
console.log(typeof NaN);        // number，Not a Number
console.log(typeof Infinity);    // number，无限大

(2) boolean —— 布尔值

console.log(typeof true);
console.log(typeof false);

(3) string —— 字符串

正常字符串

正常的字符串使用单引号或双引号包裹

var str1 = 'abc';
var str2 = "abc";
console.log(str1);             // abc
console.log(str2);          // abc
console.log(typeof 'abc');    // string
console.log(typeof "abc");    // string

转义字符

如果要输出单引号或双引号，可以使用转义字符

console.log('\'abc\''); // 'abc'
console.log("\"abc\""); // "abc"

还有其他的一些特殊字符

\        转义
\n        换行
\t         表格
\u4ed2    Unicode字符
\x41    ASCII码

多行字符串

使用~键包裹字符串，可以实现多行字符串输出

console.log(`111
222
333`);

模板字符串（新特性）

使用~键包裹字符串，可以实现调用已存在的字符串（类似EL表达式）

let name = "大白";
let age = 3;
let info = `姓名：${name}，年龄：${age}`;
console.log(info);  // 姓名：大白，年龄：3

字符串的不可变性

var stu = "student";

// 获取字符串的某个字符
console.log(stu[0]);        // s
stu[0]=1;                   // 无法修改
console.log(stu);           // 证明字符串的不可变性

字符串的属性

length：获取字符串的长度

var stu = "student";
console.log(stu.length);    // 7

字符串的方法

str.toUpperCase()：小写转大写
str.toLowerCase()：大写转小写
str.indexOf(“s”)：获取字符s在字符串str第一次出现的索引位置
str.subString(index)：字符串截取,从index索引处，截取到最后
str.subString(first,last)：字符串截取【开始索引，结束索引），包含开始索引，不包含结束索引

var stu = "student";

console.log(stu.toUpperCase());     // STUDENT
console.log(stu.toLowerCase());     // student
console.log(stu.indexOf('t'));      // 1
console.log(stu.substring(2));      // udent
console.log(stu.substring(2,5));    // ude

(4) null —— 空

console.log(typeof null);    // object

(5) undefined —— 未定义

console.log(typeof undefined);    // undefined

运算符

逻辑运算符

&&     两真且才真

||    有真或为真

!    真即假，假即真

比较运算符

=    赋值

==    等于（类型不一样，值一样，结果也是true）

===    绝对等于（类型一样，值一样，结果才是true）

在JS中，尽量使用===，而不使用==

注意：

// NaN是一个特殊的数值，他与所有数值都不相等，包括它本身，只能通过`isNaN(NaN)`判断
console.log(NaN===NaN)        // false
console.log(isNaN(NaN))        // true

// 尽量避免使用浮点数进行运算，存在精度问题!
console.log((1/3) === (1-2/3))                        // false
console.log(Math.abs((1/3) === (1-2/3))<0.0000001)    // true

数组（中括号）

JS中的数组元素可以是任意的数据类型

当数组下标越界时，报undefined，而不是报异常（JS中没有异常）

// 推荐该种方法，[]保证代码可读性
var arr = [1,2,3,"hello",null,false];
var arr2 = new Array(1,2,3,"world",null,true);
console.log(arr[3]);    // hello
console.log(arr[6]);    // undefined，数组越界不报异常，而是未定义
console.log(arr2);      // [1, 2, 3, "world", null, true]

二维数组

var arr1 = [[1,2],[3,"4"],["5","6"]];
console.log(arr1);  // 0:(2) [1, 2]     1:(2) [3, "4"]     2:(2) ["5", "6"]
console.log(arr1[1][1]);    // 4

数组的属性

arr.length：获取数组长度，改变数组长度（赋值过小，会导致元素丢失）

var arr = [1,2,3,"hello",null,false]; 
console.log(arr);       // (6) [1, 2, 3, "hello", null, false]
console.log((arr.length));  // 6
arr.length = 10;
console.log(arr);       // (10) [1, 2, 3, "hello", null, false, empty × 4]
console.log((arr.length));  // 1
arr.length = 3;
console.log(arr);       // (3) [1, 2, 3]
console.log((arr.length));  // 3

数组的方法

获取

arr.indexOf(ele)：获得ele元素在数组中第一次出现的索引

  • 注意：字符串“1” 和数字 1 是不同的

截取

arr.slice(index)：截取数组（从index下标到最后），返回一个新数组

arr.slice(first,last)：截取数组（从first到last，包括index），返回一个新数组

添加元素 / 删除元素

arr.push(ele1,ele2)：压入一个或多个新元素到尾部，返回新数组的长度

arr.pop()：弹出尾部的一个元素，返回被弹出的元素

arr.unshift(ele1,ele2)：压入一个或多个新元素到头部，返回新数组的长度

arr.shift()：弹出头部的一个元素，返回被弹出的元素

排序

arr.sort()：正序排序数组

arr.reverse()：元素反转

拼接 + 连接

arr1.concat(arr2)：将arr2数组的元素拼接到arr1数组中

  • 注意，这并不会改变原来的数组，只会返回一个新数组

arr.join(str)：打印拼接数组，使用特定字符串连接

覆盖

arr.fill(ele)：将数组所有元素都覆盖为ele

对象（大括号）

对象中的属性可以是任意类型，对象又若干个键值对组成

属性之间使用逗号分隔，最后一个属性之后不用添加逗号

var 对象名 = {
    属性名：属性值,
    属性名：属性值,
    属性名：属性值
}

javaScript中所有的键都是字符串，值是任意对象。

var person = {
    name: "大白",
    age: 3,
    subjects: ["计算机网络","数据结构与算法","和孩子一起学Python"]
};
console.log(person.name);           // 大白
console.log(person["age"]);         // 3
console.log(person.subjects);       // ["计算机网络", "数据结构与算法", "和孩子一起学Python"]
console.log(person.subjects[1]);    // 数据结构与算法

获取不存在的属性，不会报错，而是输出undefined

console.log(person.xxx);        // undefined

属性重新赋值

// 对象赋值
person.name = "小黑";
console.log(person.name);       // 小黑
console.log(person);            // {name: "小黑", age: 3, subjects: Array(3)}

添加属性，直接添加即可

person.gender = "不详";
console.log(person);     // {name: "小黑", age: 3, subjects: Array(3), gender: "不详"}

动态删除属性 delete，返回一个布尔值

console.log(delete person.age); // true
console.log(person);    // {name: "小黑", subjects: Array(3)}

判断属性是否存在该对象中 | 判断对象是否有该方法

console.log('age' in person);    // true
console.log('toString' in person);    //true，  toString()在person对象的父类中

判断一个属性是否是该对象自身拥有的

console.log(person.hasOwnProperty('toString'));     // false ,父类的
console.log(person.hasOwnProperty('age'));          // true ，自身的

流程控制

if while do…while… for

foreach循环

var arr = [111,2,3,444,222,111,555,666];
arr.forEach(function (value, index, array) {
    console.log(value + '--' + index + '--' +array);
})

for…in…循环

// for(var index in arr){}
var arr = [111,2,3,444,222,111,555,666];
for (var index in arr){
    console.log(index +'---' + arr[index]);
}
